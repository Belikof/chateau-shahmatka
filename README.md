# Château Del Mare — шахматка

Инструменты для работы с Excel-шахматкой посёлка: участки, каталог домов,
калькулятор стоимости.

## Структура

```
chateau-shahmatka/
├── data/                      рабочие файлы (.xlsx, PDF-схема, каталог JPG)
│   ├── Новая таблица.xlsx     актуальная рабочая версия
│   ├── Шахматка new.xlsx
│   ├── Шахматка (2) — эталон.xlsx
│   ├── Веселовка_Схема_всех_участков_новая (1).pdf
│   └── Каталог_extracted/     карточки домов (JPG)
├── scripts/
│   ├── lib.py                 общие утилиты (каталог, разбор участков, восстановление картинок)
│   ├── inspect_xlsx.py        осмотр структуры файла
│   ├── fill_genplan_plots.py  заполнить «Участок» на «Ген. план» из «Квартал 1»
│   └── build_calculator.py    калькулятор на «Квартал 1» (строки 29–30)
├── requirements.txt
└── README.md
```

## Установка

```bash
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
```

## Использование

Все скрипты принимают путь к .xlsx (по умолчанию `data/Новая таблица.xlsx`).

```bash
python scripts/inspect_xlsx.py "data/Новая таблица.xlsx"
python scripts/fill_genplan_plots.py "data/Новая таблица.xlsx"
python scripts/build_calculator.py "data/Новая таблица.xlsx"
```

## Важно про картинки

`openpyxl` при сохранении **теряет картинки и чертежи** (генплан и т.п.).
Поэтому правки делаются только через `lib.workbook_edit` — он автоматически:

1. делает бэкап,
2. сохраняет правки через openpyxl,
3. возвращает media/drawings из бэкапа прямой пересборкой zip,
4. удаляет бэкап.

Не сохраняйте файл «голым» `wb.save()` без восстановления — пропадёт генплан.

## Данные и правила

- **Активные улицы (в продаже):** Айвазовского, Луначарского — 127 участков.
- **Площади участков:** PDF-схема (крупно — №, мелко — м², ±5 м²).
- **Каталог домов:** 1- и 2-этажные, цены Черновая / ПЧО / Под ключ (`lib.CATALOG`).
- **Цена земли** = площадь × ₽/м² (база: мин. дом под ключ / 500 м²).
- **Угловой участок** (>500 м²): надбавка к дому = площадь × 1 000 ₽.

## Калькулятор (лист «Квартал 1», строки 29–30)

| Участок ▾ | Дом ▾ | Отделка ▾ | Способ покупки ▾ | ИТОГ |
|-----------|-------|-----------|------------------|------|

`ИТОГ = цена земли + угловая надбавка + цена дома по выбранной отделке`.

«Способ покупки» пока не влияет на сумму — нужны условия ипотеки/рассрочки
(% первого взноса, ставка, срок), чтобы считать первый платёж и график.
